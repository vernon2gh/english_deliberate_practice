## 简介

学习这个文件输入/输出内容，作为模板，然后读取 sentences.txt 文件里面的所有提供的句子，
输出相同格式的内容，将内容写到 output.txt 文件即可。

## 注意：

1. 单词(a/bit/a bit/cold)组合成词组(a bit cold)后，一定要提供一次练习(a bit cold)，
   再与前面词组组合起来(The weather was a bit cold)

## 输入句子：

hello world. 您好。
How are you? 你好吗？
We need to address the issue of global warming. 我们需要解决全球变暖的问题。
This tool is perfectly fit for purpose. 这个工具非常适合其用途。
Today I ate the green pepper rice.
The weather was a bit cold
